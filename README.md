# Dockerize Node 

## Requisitos

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

## Usage

up container
```bash
docker-compose up -d
```
list container
```bash
docker-compose ps
```
show container logs
```bash
docker-compose logs app
```

[http://localhost:3000](http://localhost:3000).
